# OpenClaw Skills

Reusable OpenClaw skills used in my local agent setup.

## Skills

- `skills/mistral-pdf-ocr` - Run Mistral OCR on scanned PDFs and generate Markdown, JSON, and searchable PDFs with preserved page images.

## Safety

This repository is intended for public skill code only.

- Do not commit API keys, tokens, secrets, private memory files, or OCR output files.
- Keep generated files under local `temp/` or another ignored output directory.
- Review each skill before publishing it here.
