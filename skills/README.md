# Skills

Each subdirectory contains one reusable OpenClaw skill.

## Available Skills

- `mistral-pdf-ocr`: Run Mistral OCR on scanned PDFs and generate Markdown, JSON, and searchable PDFs.
- `vgage-technical-agreement-extractor`: Extract VGAGE technical agreements into a configured Notion analysis database with project-progress routing safeguards.
- `vgage-project-progress`: Update VGAGE weekly project progress from PPT files, manage focused projects, and link progress reminders with ECM-style folder monitoring.
- `vgage-measurement-probe-binding-check`: Check and fill VGAGE Measurement-to-Probe bindings after creating, migrating, or modifying inspection items.
- `vgage-probe-dependencies-check`: Check and fill VGAGE computed Probe Dependencies after creating, migrating, or modifying Probe formulas.
